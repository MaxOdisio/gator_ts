# Gator - CLI RSS Feed Aggregator 🐊

Versione Typescript di Gator, un aggregatore di feed RSS basato su riga di comando scritto in Go. Il progetto permette di gestire utenti, seguire feed RSS e sincronizzare i contenuti all'interno di un database PostgreSQL.
